# UPI Fraud Detection using Machine Learning

##  Overview

This project detects fraudulent UPI transactions using machine learning techniques.

## Objective

To classify transactions as fraud or non-fraud based on transaction features.

##  Dataset

The dataset contains transaction details such as amount, transaction type, device changes, and IP changes.

## Techniques Used

* Data preprocessing (encoding + scaling)
* Logistic Regression
* Random Forest

##  Results

Random Forest achieved higher accuracy(93.9%) compared to Logistic Regression(91.7%).

##  Features

* Confusion matrix
* Classification report
* Feature importance analysis
